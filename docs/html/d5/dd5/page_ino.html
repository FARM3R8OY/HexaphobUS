Introduction

Arduino code designed for intuitive control and monitoring of the HexaphobUS robot.

The servomotors use an I2C-bus controller (Sunfounder PCA9685) to communicate, with a 12-bit resolution (4096 steps).

Dependencies

This library depends on Adafruit PWM Servo Driver Library being present on your system. Please make sure you have installed the latest version before using this library.
