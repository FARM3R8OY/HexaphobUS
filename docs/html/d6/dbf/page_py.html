Introduction

Python code designed for intuitive control and monitoring of the HexaphobUS robot.

The user interface works on a on-board computer (Raspberry Pi 3B) to communicate, with a standard modulation rate of 9600.

Dependencies

This package depends on <a href="https://pypi.org/project/PyQt5/> PyQt5</a>, <a href="https://pypi.org/project/pyserial/">pyserial, and other standard modules being present on your system. Please make sure you have installed the latest versions before using this interface.
